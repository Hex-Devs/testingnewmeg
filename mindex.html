Imagine a bot...
Megumin is a Fun & Moderation bot. It has a dashboard, that you can open in Discord by using command. If you don't like any of the modules, you could just turn it off! Bot has mini-games like rock-paper-scissors. Also you can chat with a custom AI. And of course there are moderation comands like ban and mute. You can set up custom audit channel for Megumin, so it will be able to log there actions on the server. [Date of release - 30th June2021]
